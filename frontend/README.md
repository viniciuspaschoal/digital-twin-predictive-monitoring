# Interface do Usuário (Frontend)
Este diretório deve conter os arquivos do **Dashboard**.
**Responsabilidades:**
- Exibir os dados em tempo real vindos do MQTT/API.
- Mostrar gráficos históricos de vazão e temperatura.
- Exibir alertas visuais de manutenção preditiva.
